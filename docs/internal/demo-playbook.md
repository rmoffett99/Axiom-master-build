# AXIOM Demo Playbook

## 1) Opening (30 seconds)

"Axiom is a decision intelligence layer. It doesn't replace people — it makes decisions explainable, reviewable, and auditable."

## 2) Dashboard (2 minutes)

- Show active decisions
- Highlight risk indicators
- Emphasize: this is decision state, not tasks

## 3) Decision Walkthrough (3 minutes)

- Context → options → risk → outcome
- Emphasize why this decision was chosen
- Emphasize traceability

## 4) Alerts (2 minutes)

- Show proactive risk escalation
- Emphasize prevention vs reaction

## 5) Audit Trail (2 minutes)

- Show immutable timeline
- Emphasize compliance and accountability

"Who decided what, when, and why."

## 6) Close (1–2 minutes)

"Axiom gives leadership confidence in decisions — even months later."

## Sales Readiness Checklist

Before any demo:

### Routing & Stability
- [ ] Confirm baseline metrics unchanged
- [ ] Zero 404s in last 24 hours
- [ ] Run routing regression test (A–D)

### Demo Environment
- [ ] axiom-demo org loads cleanly
- [ ] Demo data intact
- [ ] Demo banner visible
- [ ] Destructive actions disabled

### Performance
- [ ] Dashboard load within baseline
- [ ] No console errors
- [ ] No broken links

### Narrative
- [ ] Demo script reviewed
- [ ] Talking points aligned
- [ ] No unfinished features shown

**RULE:** If ANY item fails, demo is postponed. Issue fixed first.
